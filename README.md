# COGNITIVE GAME
## SOBRE O PROJETO
O declínio cognitivo leve afeta milhões de idosos globalmente, reforçando a necessidade de intervenções digitais acessíveis e escaláveis alinhadas às diretrizes de Atenção Integrada para a Pessoa Idosa da Organização Mundial da Saúde. Com o propósito de contribuir para o desenvolvimento de tecnologias voltadas à saúde dessa população, foi desenvolvido o Cognitive Game, um serious game web que utiliza linguagens de desenvolvimento web e integra minijogos validados a uma narrativa acessível para estimulação de atenção, raciocínio e memória em idosos com declínio cognitivo leve. Fundamentado em documento de design de jogo e em revisão sistemática (Organização Mundial da Saúde 2024, Stern 2012), o jogo prioriza usabilidade (interfaces simples, feedback imediato) e baixo custo, demonstrando viabilidade técnica via testes internos de funcionalidade e compatibilidade. Os resultados preliminares indicam que o Cognitive Game possui potencial para subsidiar futuros ensaios clínicos randomizados voltados à validação de seus possíveis efeitos sobre a estimulação cognitiva e a neuroplasticidade. 

## TECNOLOGIAS UTILIZADAS

- Criação de imagens: Microsoft Bing;
